# CNN-from-Scratch Experiments

This folder contains CNN-from-scratch experiments focused exclusively on tomato leaf disease classification.

## Structure

- `experiments/`: Contains individual experiment implementations
- `results/`: Contains output results from each experiment

## Experiments

1. **01_baseline_cnn**: Basic CNN architecture baseline
2. **02_optimized_cnn**: Optimized CNN with improvements
3. **03_cnn_feature_extractor**: CNN backbone for feature extraction
